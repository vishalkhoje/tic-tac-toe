
# TIC-TAC-TOE

## Run Locally

Clone the project

```bash
  git clone https://github.com/vishalkhoje/tic-tac-toe.git
```

Go to the project directory

```bash
  cd tic-tac-toe
```

Install dependencies

```bash
  npm install
```

Start the React App

```bash
  npm start
```
